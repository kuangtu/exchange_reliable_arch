简介：由深圳证券通信有限公司工程师发表在[《交易技术前沿》总第三十二期文章（2018年9月）](https://istock.ssetech.com.cn/wiki/doku.php?id=service:techmag:201809_032:start) 上面。

能够对于交易相关系统“分布式”转型之前的架构进行学习和了解。
